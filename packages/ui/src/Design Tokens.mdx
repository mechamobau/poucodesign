# Design Tokens

## Introdução

Os design tokens são as variáveis semânticas de estilo como cores, espaçamento, tipografia, bordas, etc, espelhados entre design e tecnologia que nos permitem escolher decisões de design com consistência e escala.

Essencialmente, os tokens correspondem à fundação sob a qual todos os componentes e interfaces serão construídas.
